My name is Drew Harris. I'm a musician and software developer living in Victoria, Canada. I studied Electrical Engineering at the University of Victoria from 2008 to 2013. I've worked as a JavaScript developer at exfm in New York, a Java developer for Alcatel-Lucent in Ottawa, and as a Python developer for the University of Victoria in Victoria.

Through this employment experience, class work and my own projects, I've learned to write client- and server-side JavaScript, C++, Python and Java code to perform wide-ranging tasks, from large enterprise web applications to iOS applications that provide visual accompaniment to a live musical performance. I build physical devices to provide new ways to interact with music in a live performance setting, including an electronic light array that can be sequenced as an instrument.

Along with software and hardware development, I compose and produce electronic music under several monikers, including Germany Germany, radioseven and Toy Camera. I've performed in Vancouver, New York City, Paris, and Berlin, among other locations, and I've released a total of 8 albums independently.
